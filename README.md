# Personal-discord-bot
___
This is a discord app (a discord bot that requires user install, not server install)

This program only requires basic command perms, very simple, and no "dangerous" permissions

___
Commands list

* /stop = only works if bot owner as you have to have the right discord id. Disables the bot

* /hello = basic input output for testing if bot is running

* /wordlehelp = this command uses easy ui navigation in order to help solve the wordle

* /pwn = troll command to make the specifyed user thing theyre getting hacked(all info can be found on discord easily)\

* /urgent =  sends a mention of the sppecifyed user every 1-2 minutes utill the user presses the button to turn off or 15 min pass
